## 🎬 Getting Started

### Step 1: Setup (5 minutes)
```bash
cd e:\HTML CSS JAVASCRIPT\Hms1\Hms1
npm run build
php artisan serve
```
👉 See [QUICK_START.md](QUICK_START.md#-5-minutes-setup)

### Step 2: Login & Explore (5 minutes)
1. Visit `http://localhost:8000`
2. Login with test credentials
3. Click "Book Appointment"
👉 See [QUICK_START.md](QUICK_START.md#-default-test-data)

### Step 3: Book an Appointment (2 minutes)
1. Browse available doctors
2. Click "Book Appointment"
3. Fill form & confirm
👉 See [QUICK_START.md](QUICK_START.md#--booking-status-flow)
